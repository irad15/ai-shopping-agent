# AI Shopping Agent - Frontend

A premium, multi-turn AI shopping agent interface built with Next.js and Assistant-UI.

## ✨ Features
- **Generative UI**: Real-time product cards rendered directly in the chat thread.
- **Multi-turn Context**: Full conversation history support, including tool call recovery.
- **Premium Design**: Dark mode aesthetic with Zinc/Indigo accents and smooth Framer Motion transitions.
- **Robust Validation**: Zod-powered runtime validation for all streamed tool payloads.

## 🛠 Tech Stack
- **Framework**: [Next.js 16.1.6](https://nextjs.org/)
- **UI Architecture**: [Assistant-UI](https://assistant-ui.io/)
- **Streaming**: [Vercel AI SDK](https://sdk.vercel.ai/)
- **Validation**: [Zod](https://zod.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Ensure the backend is running at `http://localhost:8000`.
4.  Launch the development server:
    ```bash
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) in your browser.
